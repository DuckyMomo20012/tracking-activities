- Nếu chưa có python thì chạy file "installPython.bat"
- Để chạy chương trình thì có thể chạy 3 file sau:
  +"main.exe"
  +"main.py" (Phải ở trong môi trường virtualvenv tên là 'env')
  +"runWin.bat"
- Để uninstall python thì chạy file "uninstallPython.bat"
- Khi chạy upload thì tất cả file trong thư mục "pictures" sẽ được upload lên thư mục tên GGDrive
